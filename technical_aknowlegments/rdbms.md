# Что такое RDBMS

**RDBMS (Relational DataBase Management System)** — это система управления реляционными базами данных.  
В ней данные хранятся в виде таблиц (строк и столбцов), которые могут быть связаны между собой через ключи.

## Основные характеристики

- **Табличная модель данных** — информация хранится в таблицах (relations).
- **Связи между таблицами** — обеспечиваются с помощью первичных и внешних ключей.
- **SQL** — основной язык работы с данными (запросы, модификация, управление транзакциями).
- **ACID-свойства** — гарантируют надежность транзакций:
  - **A**tomicity — атомарность  
  - **C**onsistency — согласованность  
  - **I**solation — изоляция  
  - **D**urability — долговечность
- **Нормализация данных** — уменьшение избыточности и улучшение целостности данных.

## Примеры RDBMS

- PostgreSQL
- MySQL / MariaDB
- Oracle Database
- Microsoft SQL Server
- SQLite

## Когда использовать RDBMS

- Необходима строгая структура данных.
- Требуются надежные транзакции и ACID-гарантии.
- Важны связи между сущностями (отношения).
- Нужна сложная выборка и аналитика на основе SQL.
